# FASE 2 — PRD (Product Requirements Document)

## Instruções Executáveis para IA
- Este documento é a **fonte única da verdade**
- Nenhuma funcionalidade fora deste escopo deve ser implementada
- Em caso de dúvida, priorize este arquivo

(Conteúdo completo do PRD deve ser lido a partir do documento principal)

## Objetivo
Documentar de forma clara e executiva os requisitos funcionais, não funcionais e regras de negócio da Calculadora Previdenciária (V1).

## Conteúdo do PRD
- Visão geral da solução
- Objetivos de negócio e KPIs
- Escopo da V1 e roadmap V2
- Personas
- Jornada do usuário
- Dados de entrada e saída
- Regras de negócio (cálculo)
- Backoffice administrativo
- Stack tecnológica
- Riscos e mitigações

## Papel do PRD
- Fonte única da verdade (single source of truth)
- Base para desenvolvimento técnico
- Documento de alinhamento com stakeholders
- Referência para evolução futura (V2)

## Status
PRD aprovado e validado para execução técnica.
